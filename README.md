# pwa-parking-relais-lyon

Récupération des disponibilités en temps réel des P+R Lyon

## Accès à l’API Data Grand Lyon (important)

⚠️ La création d’un compte via "Grand Lyon Connect" permet de se connecter
au portail data.grandlyon.com mais ne suffit PAS pour accéder aux API Datapusher.

✅ Pour activer l’accès API :
1. Aller sur https://data.grandlyon.com
2. Utiliser la fonction "Mot de passe oublié" spécifique au portail data
3. Définir un mot de passe propre à data.grandlyon.com
4. Utiliser cet email + mot de passe pour l’authentification Basic Auth
